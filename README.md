## Installation

```bash
$ npm install
$ docker-compose up -d --build
```
After this, you should be able to access the database phpPgAdmin on localhost:5050 with the default credentials:
email = admin@example.com
password = admin_pass

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
